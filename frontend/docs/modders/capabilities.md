# 当前能力

LongLive Lib 现在对 Modder 更有价值的内容，主要在这几类能力上。

## 运行时诊断

目前已经有比较完整的调试入口和运行时跟踪能力，包括：

- 战斗 Tracer
- 地图 Tracer
- Host / Bridge 诊断信息
- Native Probe

如果你在排查战斗、场景切换、地图行为或者 Host 状态，这部分已经能提供不少帮助。

## SceneRouting 主线

现在这条主线的目标，是把原本零散的场景切换行为整理成一套更稳定的中间层。

这部分已经不只是简单包一层 `loadMapScenes(...)`，而是在往更像运行时服务的方向走。

## 自定义地图前置能力

地图概览、地图注册、运行时激活、Host 绑定这些内容，现在都已经开始成形。

它们还没有完全变成稳定 API，但已经不是零散试验状态了。

## JSON 与 bridge 方向

这部分目前仍在早期阶段。

现在已经有：

- JSON demo 样例
- Next bridge demo 样例

但它们还不适合被当成稳定公开接口来依赖。

## 你现在可以把它当成什么

更实际一点的说法是：

- 它已经可以当成调试和运行时能力的集合来用
- 某些主线方向已经可以围绕它继续开发
- 但公开 API 还没有完全定型，接入时最好预留调整空间
