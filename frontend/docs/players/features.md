# 功能说明

LongLive Lib 现在面向玩家最直接的内容，主要集中在下面这些地方。

## 批量使用物品更顺手

当你一次性吃很多丹药、妖丹，或者连续使用大量可消耗物品时，游戏原本很容易卡住一段时间。

LongLive Lib 现在会把这条流程拆开处理，让结算分帧进行，尽量减少明显卡顿。

目前这部分还包括：

- 长按右键打开数量选择
- 中键也能打开数量选择
- 大量使用后尽量把提示合并，而不是整屏刷满

## 提示不再一直挡视野

游戏里有些提示一多起来，会在屏幕右侧堆很久，甚至退回主菜单后还在继续弹。

LongLive Lib 会尽量合并重复提示，并清理积压太多的提示，让信息更容易看。

## 战斗里少一些离谱卡顿

有些高伤、多段的战斗结算，本来会出现目标已经死了，后面的伤害还在继续算的情况。

这类情况往往会顺带带来掉帧、音效叠在一起、表现异常之类的问题。

LongLive Lib 现在会做一层保护，尽量减少这种无意义的后续结算。

## 部分黑屏过场会更快一点

有些共享黑屏和场景切换本来节奏比较拖，LongLive Lib 会对其中一部分做加速。

这部分不是所有路径都强改，主要目标还是让常见流程别显得太磨人。

## 主菜单里可以确认有没有正常加载

主菜单里的 `LongLive Lib` 入口不只是给开发者看的。

对普通玩家来说，它也能帮你确认：

- Mod 有没有加载成功
- Host / Bridge 当前是不是正常
- 某些兼容层有没有生效

## 现在要注意的事

- 这是一个还在继续开发的项目，后面功能还会继续变动
- 启动加载时间比平时长一点是正常现象
- 更推荐和 `Next` 一起使用，兼容性会更稳一些
- 如果你不喜欢某些优化，通常可以在 `F1` 的配置界面里关闭
