# Current Capabilities

LongLive Lib is already useful to modders in a few concrete areas.

## Runtime diagnostics

Current diagnostics include:

- battle tracing
- map tracing
- host and bridge diagnostics
- native probe support

## Scene-routing main line

Scene-routing is being shaped as a real middle layer rather than a thin wrapper around raw host calls.

## Custom-map groundwork

Map overview integration, registration, runtime activation, and host-binding work are all taking shape.

## JSON and bridge direction

These areas are still early, but sample content already exists for experimentation.
