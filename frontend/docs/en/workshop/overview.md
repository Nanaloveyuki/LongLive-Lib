# Workshop Notes

This page is the short player-facing overview for the workshop version of LongLive Lib.

If you install it now, the most visible changes are practical optimizations and a clearer runtime status entry in the main menu.

## What it is

LongLive Lib is an experimental runtime library built around the `BepInEx` and `Next` ecosystem.

For players, the useful part today is simple: it already improves several rough edges in the game and makes runtime status easier to inspect.

Using it together with `Next` is still the safer option for compatibility.

Slightly longer startup is normal because the mod loads extra runtime content.

## What it currently improves

- smoother bulk item use
- merged and cleaned-up pop tips
- battle-side overkill protection
- selected fade-heavy transitions
- pinyin support in TuJian search
- a visible main-menu status entry
